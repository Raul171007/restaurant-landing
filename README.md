# La Piazza — Landing Page para Restaurante Italiano

Landing page moderna y totalmente responsive para un restaurante italiano, desarrollada con HTML, CSS y JavaScript puro. Sin frameworks ni dependencias — código limpio y rápido.

## Demo en vivo

[Demo](https://restaurant-landing-ashen.vercel.app/)

## Características

- Diseño totalmente responsive (móvil, tablet, escritorio)
- Animaciones suaves al hacer scroll con Intersection Observer API
- Navbar fija con efecto al hacer scroll
- Formulario de reserva con mensaje de confirmación
- Google Fonts (Playfair Display + DM Sans)
- Iconos con Font Awesome
- Variables CSS personalizadas
- Código limpio, organizado y comentado

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

## Lo que aprendí

- Uso de la Intersection Observer API para animaciones al hacer scroll
- CSS Grid y Flexbox para layouts complejos
- Cómo estructurar una landing page profesional
- Diseño responsive con media queries
- Variables CSS para mantener consistencia visual

## Sobre el proyecto

Este proyecto fue desarrollado como pieza de portfolio para demostrar habilidades de desarrollo frontend. Simula un proyecto real de cliente — una landing page para un restaurante familiar italiano — y cubre los requisitos más habituales en este tipo de trabajo: diseño atractivo, compatibilidad móvil, formulario de contacto y carga rápida.
